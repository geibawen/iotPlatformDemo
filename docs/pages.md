# 页面功能说明

## 1. 概览页（Dashboard）

**路径**：`/`

展示平台整体概况：
- **统计卡片**：产品数量、插件数量、推送规则数、推送日志数
- **产品列表**：最近创建的产品快速预览
- **最近日志**：最新推送日志记录

---

## 2. 产品管理

### 2.1 产品列表（ProductList）

**路径**：`/products`

- 表格展示所有产品，支持按名称搜索
- 操作：创建、编辑、删除、查看详情
- 产品表单（Modal）：名称、分类、连接方式、图片 URL、描述

### 2.2 产品详情（ProductDetail）

**路径**：`/products/:id`

顶部展示产品基本信息和状态，下方 Tab 页切换：

| Tab | 内容 |
|-----|------|
| 基本信息 | 产品字段的只读展示 |
| 属性（Property） | 物模型属性表格，支持 CRUD |
| 动作（Action） | 物模型动作表格，支持 CRUD |
| 服务（Service） | 物模型服务表格，支持 CRUD |
| 关联插件 | 展示关联到该产品的所有插件 |

---

## 3. 物模型编辑

物模型编辑器嵌入在产品详情页的 Tab 中，分为三部分：

### 3.1 属性编辑（PropertyTable + PropertyForm）

属性表格列：标识符、名称、数据类型、读写模式、单位、是否必填

**属性表单**是最复杂的表单，根据选择的数据类型动态渲染不同的 specs 字段：

| 数据类型 | 动态字段 |
|---------|---------|
| int / float | 最小值、最大值、步长、单位 |
| bool | 真值标签、假值标签 |
| string | 最大长度 |
| enum | 枚举值列表（动态增减行） |
| struct | 结构体字段列表（动态增减行） |
| array | 数组元素类型 |

### 3.2 动作编辑（ActionTable + ActionForm）

动作表格列：标识符、名称、输入参数数、输出参数数

表单包含：
- 基础信息：标识符、名称、描述
- 输入参数列表（Form.List 动态增减）
- 输出参数列表（Form.List 动态增减）

### 3.3 服务编辑（ServiceTable + ServiceForm）

服务表格列：标识符、名称、关联属性数、关联动作数

表单包含：
- 基础信息：标识符、名称、描述
- 关联属性：多选下拉（从当前产品属性中选择）
- 关联动作：多选下拉（从当前产品动作中选择）

---

## 4. 插件管理

### 4.1 插件列表（PluginList）

**路径**：`/plugins`

- 表格展示插件名称、平台、关联产品（Tag 标签）、状态、更新时间
- 操作：创建、编辑、删除、查看详情
- 插件表单（Modal）：名称、描述、平台选择、关联产品（Transfer 穿梭框）

### 4.2 插件详情（PluginDetail）

**路径**：`/plugins/:id`

顶部展示插件信息和启用/停用状态，下方 Tab：

| Tab | 内容 |
|-----|------|
| 基本信息 | 插件字段的只读展示 |
| 版本管理 | 版本列表和发版操作 |

### 4.3 版本管理（VersionManager）

版本表格列：版本号、文件名、文件大小、状态、发布说明、创建时间

功能：
- **新建版本**：填写版本号、发布说明，上传插件包文件（真实文件上传，`multipart/form-data`，最大 50MB）
- **状态流转**：draft（草稿）→ testing（测试中）→ released（已发布）
- 每个状态只能向下一个状态推进

---

## 5. 推送配置

### 5.1 推送规则（PushRules）

**路径**：`/push/rules`

- 表格展示规则列表，支持 Switch 快速开关
- 规则表单：
  - 规则名称
  - 关联产品（下拉选择）
  - 触发类型（设备上线/离线/属性变更/事件上报）
  - 触发条件（动态条件行：字段 + 操作符 + 值）
  - 推送通道（下拉选择已创建的通道）
  - 消息模板（下拉选择已创建的模板）

### 5.2 推送通道（PushChannels）

**路径**：`/push/channels`

- 表格展示通道名称、类型（Tag 颜色区分）、状态、操作
- **连接测试**：点击"测试"按钮模拟连接测试（2 秒延迟 + 成功提示）
- 通道表单根据类型动态渲染不同配置字段：
  - **HTTP 回调**：URL、Method、Headers、Secret
  - **MQTT**：Broker 地址、端口、Topic、用户名、密码
  - **Kafka**：Brokers、Topic、Group ID
  - **MNS**：Endpoint、AccessKeyId、AccessKeySecret、Topic

### 5.3 消息模板（MessageTemplates）

**路径**：`/push/templates`

- 表格展示模板列表
- 模板表单：名称、描述、格式（JSON/TEXT）、模板内容
- **变量插入**：点击变量按钮自动插入占位符到内容编辑器
- **模板预览**：查看模板内容（代码高亮展示）

### 5.4 推送日志（PushLogs）

**路径**：`/push/logs`

- 表格展示推送记录，支持按状态筛选（全部/成功/失败）
- **详情抽屉（Drawer）**：查看完整的请求体、响应体、错误信息
- 操作：刷新、清空日志

---

## 6. 角色模拟与切换

### 6.1 右上角角色切换

各角色布局右上角头像下拉菜单支持角色切换：

- 切换到开发者：`/dashboard`
- 切换到测试人员：`/tester/plugins`
- 切换到系统管理员：`/admin/product-categories`

### 6.2 测试人员工作台（TesterLayout）

**路径**：`/tester`

- 插件测试：`/tester/plugins`
- 白名单管理：`/tester/whitelist`

### 6.3 系统管理员工作台（AdminLayout）

**路径**：`/admin`

- 产品分类管理：`/admin/product-categories`
- 支持产品分类增删改查
